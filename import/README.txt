Папка import — для файлов с экзамена (Excel из Приложения 2).

На экзамене:
1. Скопируй сюда Excel-файлы из архива задания.
2. Импортируй данные в SSMS в базу ShoesStore (таблицы Users, Tovar, Order и т.д.).
3. Фото товаров положи в WinFormsApp5\Images (имена как в столбце Photo, например 1.jpg).
4. Строку подключения поменяй в LoginForm.cs и DataSet.xsd (Data Source=...).

Стек: Visual Studio, .NET Framework 4.8, SQL Server + SSMS.
